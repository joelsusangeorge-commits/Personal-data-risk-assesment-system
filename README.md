# Personal-data-risk-assesment-system
A Java-based privacy risk assessment tool that analyzes user security habits and displays a privacy risk dashboard.
The application helps users understand how their everyday online behavior can affect their personal data privacy and security.

Project Overview

In today's digital world, users interact with multiple online platforms such as social media, banking services, and communication applications. Many users unknowingly follow unsafe digital practices like reusing passwords, connecting to unsecured public WiFi networks, or sharing sensitive information.

This project provides a simple risk assessment tool that analyzes user responses and generates a privacy risk score and dashboard to help users understand their level of exposure to privacy risks.

Features
One question at a time questionnaire
Six security-related questions
Risk calculation based on user behavior
Privacy risk dashboard output
Color-changing risk meter
Categorized risk analysis

How It Works
The system asks the user a series of security-related questions.
The user selects an answer for each question.
Each answer is assigned a risk score.

Risk scores are categorized into:
Authentication Risk
Social Media Risk
Network Risk
Financial Risk

The system calculates a total risk score.
The final result is displayed in a dashboard with a visual risk meter.
